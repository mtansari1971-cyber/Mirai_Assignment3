# 🌌 AI Multiverse - The Memory Vault

A conversational AI chatbot built with **Streamlit** and **Google Gemini API**. This project demonstrates how to create a **stateful chatbot** using **Streamlit Session State**, allowing the application to remember chat history even after the app reruns.

---

##  Project Overview

The AI Multiverse allows users to chat with different AI personalities while maintaining a continuous conversation. Unlike a stateless chatbot, this application stores previous messages using `st.session_state`, ensuring that chat history remains visible throughout the session.

---

##  Features

*  Interactive chatbot built with Streamlit
*  Powered by Google Gemini 2.5 Flash
*  Multiple AI personalities
*  Chat interface using `st.chat_input()`
*  Chat history stored using `st.session_state`
*  Remembers previous messages during the session
*  Clear Chat functionality
*  Fast AI responses with loading spinner
*  Clean and user-friendly interface
